RTL8192CU USB Driver
====================

Realtek WLAN USB Dongle Linux driver

Comaptible Linux Kernel versions: Linux Kernel versions 2.6.18~3.9

Realtek Base Version 3.4.4_4749

http://www.realtek.com/downloads/
